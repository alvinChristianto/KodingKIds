# KodingKids - made with Astroship - Astro SAAS Starter Website Template

KodingKids is place where kids can learn Computational Code in easy and fun way for kids age 4 up to 12.
Astroship is a free starter astro website template for saas, startups, marketing websites, landing pages & blogs. Built with Astro & TailwindCSS.

This Free Template is sponsored by [Web3Templates](https://web3templates.com)
